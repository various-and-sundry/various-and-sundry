LAMUEL

LAMUEL was may college capstone project. It uses voice recognition and a language model to respond naturally to speech. It also has a camera that it uses for face tracking and object detection. Its head can pitch and yaw to track objects, and it uses an Nvidia Jetson board to run the language model.
