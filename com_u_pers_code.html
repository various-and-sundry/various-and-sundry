COM_U_PERS_CODE

COM_U_PERS_CODE (Computer User Personality Code) is a fun way of conveying your digital preferences through a terse string of characters. Inspired by Geek Code, COM_U_PERS_CODE encodes the answers to certain predefined questions about ones preferences and abilities. It can include information about hobbies, programming languages, social media, and more. Anyone can generate a COM_U_PERS_CODE to express their own unique preferences and abilities.

The COM_U_PERS_CODE Generator is still in alpha testing, and the COM_U_PERS_CODE Parser has not yet been released. In these early stages of development, many major changes are still to be made. If you have any suggestions, please file an issue on the various-and-sundry.com GitHub repository. Suggestions of any kind are welcome.
