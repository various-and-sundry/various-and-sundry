Computer User Profile Code

Computer User Profile Code is a fun way of conveying your digital preferences through a terse string of characters. Inspired by Geek Code, Computer User Profile Code encodes the answers to certain predefined questions about ones preferences and abilities. It can include information about hobbies, programming languages, social media, and more. Anyone can generate a Computer User Profile Code to express their own unique preferences and abilities.

The Computer User Profile Code Generator can be used to create a profile code, and the Computer User Profile Code Parser can be used to decode a profile code. If you have any suggestions regarding how they can be improved, please file an issue on the various-and-sundry.com GitHub repository. Suggestions of any kind are welcome.
